### Installing Kecal Font

**For Windows:**
1. Navigate to the folder where you downloaded the Kecal font file.
2. Right-click on the `.otf` file.
3. Select "Install" from the context menu. 
4. The font will now be available in your applications.

**For macOS:**
1. Go to the folder with the Kecal font file.
2. Double-click the `.otf` file.
3. This will open Font Book. Click "Install Font" at the bottom right.
4. The font will now be available in your applications.

**For Linux:**
1. Open the terminal.
2. Move the `.otf` file to your local font directory:
   ```
   mv ~/Downloads/Kecal.otf ~/.fonts/
   ```
3. Update the font cache:
   ```
   fc-cache -f -v
   ```
4. The font will now be ready for use in your applications.
